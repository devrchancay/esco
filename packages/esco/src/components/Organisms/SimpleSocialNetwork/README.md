# SimpleSocialNetwork()

Hero for portfolio or business Webpage.

example

```jsx
<SimpleSocialNetwork
  title="Business title or portfolio"
  subTitle="Business slogan or portfolio"
  socialNetworks={[
    {
      name: "Facebook",
      url: "https://facebook.com"
    },
    {
      name: "Twitter",
      url: "https://twitter.com"
    }
  ]}
/>
```
